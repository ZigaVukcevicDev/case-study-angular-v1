UI-Router

UI-Router is the defacto standard for routing in AngularJS.

We are using .config method to set up what url should use which template (or. template file).


    angular.module('uiRouterExample', [ui.router])
        .config(function($stateProvider, $urlRouterProvider) {
            $stateProvider
                .state('home', {
                    url: '/home',
                    templateUrl: 'templates/home.html'
                }
            );
        });

We can redirect all routes which are not available to just go to home route


    angular.module('app')
        .config(function($stateProvider, $urlRouterProvider) {
            $urlRouterProvider.otherwise('/home');
    });

Using links


    <a ui-sref="home">Home</a>

For outputting content we are using directive.


    <ui-view></ui-view>
